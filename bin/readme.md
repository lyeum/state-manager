# project 스크립트 사용법

> 코드 작성 및 검사할 때 자잘한 설정 없이 바로 쓰는 커스텀 스크립트

## 스크립트 목적

- 일일히 명령어 치지 않고 코드 스타일 검사, 프리커밋, 깃허브 CI 통과를 미리 해보는 스크립트
- 코드 푸시 전, 한번씩 돌려보면 불필요한 리팩터링을 줄일 수 있음

## 사용법

- 프로젝트 루트에서 다음 명령어 실행

```bash
chmod +x ./bin/project
```

- 이후 다음과 같은 형태로 실행

```bash
./bin/project lint       # 전체 파이썬 코드 검사 및 수정
./bin/project pre-commit # 전체 파일에 대한 프리커밋 검사 및 수정(안되는거 있음)
./bin/project ci-dev     # 깃허브 액션으로 돌아갈 ci-dev 시뮬레이션
```
